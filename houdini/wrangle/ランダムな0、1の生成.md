```
int zero_one = floor(rand(@ptnum + chf("seed")) - chf("offset") + 0.5);
```
`chf("seed")`は`0 - 1`
`chf("offset")`は`0 - 0.4999`
に制限をつけておく

<br><br>
以下は
```
i@zo = floor(rand(@ptnum + chf("seed")) - chf("offset") + 0.5);
@P.y += @zo;
```
`seed : 0.142`、`offset : 0.304`
の実行結果
